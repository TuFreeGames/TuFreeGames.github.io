# One-Key Tribulation Assets

Place the character spritesheet here:

- `assets/sprites/hero-walk-spritesheet.png`

`game.js` already loads this single spritesheet and plays it at 12 FPS.
No need to split into multiple PNG files.

Expected layout:
- 6 columns x 5 rows
- 30 frames total
- playback order: left to right, top to bottom
